## Описание
Это приложение позволяет пользователям искать пользователей на GitHub, добавлять их в избранное и сохранять информацию в JSON-файле.

## Как использовать API
Для поиска пользователей используется GitHub API. Приложение отправляет запрос на `https://api.github.com/users/{username}`, где `{username}` - это имя искомого пользователя.

## Примеры использования
1. Откройте приложение.
2. Введите имя пользователя в текстовое поле.
3. Нажмите кнопку "Поиск пользователя".
4. Если пользователь существует, его имя и ссылка будут отображены в списке. 
5. Для добавления в избранное выберите пользователя в списке и нажмите "Добавить в избранное".
